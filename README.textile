h1. Tab Sugar's Website Template

h2. About

"Tab Sugar":http://tabsugar.com/ is a Google Chrome extension that helps managing your Google Chrome tabs by moving them into groups.

This Website Template has been designed for its own needs, but is generic enough to be used and reused by anyone else for any other product.

Please simply keep a reference to the author website at the bottom of the page: http://arnley.com/.
Its style can be modified but should remain visible.
